This micro website is a PWA compatible website that controls a VPower EVSE over web BLE.
VPower EVSE is a EVSE designed to be low cost and minimalistic while still retaining perfect function and conforming to all laws in the EU
The website is opensource as there is no server side code, clone it, mod it, do whatever you want but please send a copy to me so i can add improvments to this website.
